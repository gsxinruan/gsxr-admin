## 安装项目依赖

推荐使用 `yarn`：

```shell
yarn
```

解决 `node-sass` 网络慢的方法：

```shell
yarn config set sass-binary-site http://npm.taobao.org/mirrors/node-sass
```

## 运行应用程序

安装过程完成后，运行以下命令启动服务。您可以在浏览器中预览网站 [http://localhost:9000](http://localhost:9000)

```shell
yarn dev
```
